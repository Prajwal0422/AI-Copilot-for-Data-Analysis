# Animation Guide

## Entrance Animations
- Navigation: y: -20 → 0, duration: 0.6s
- Left sidebar: x: -50 → 0, delay: 0.2s
- Center: y: 50 → 0, delay: 0.3s
- Right sidebar: x: 50 → 0, delay: 0.4s

## Hover Effects
- Scale: 1.02 - 1.05
- Translate: y: -2 to -4px
- Duration: 0.3s

## Workflow Animations
- Active step: scale: 1.2
- Complete step: green gradient
- Connector line: width: 0 → 100%

## Typing Indicator
- 3 dots pulsing
- Scale: 1 → 1.5 → 1
- Staggered delay: 0.2s

## Background Orbs
- Duration: 20-25s
- Scale: 1 → 1.2 → 1
- Opacity: 0.2 → 0.5 → 0.2
