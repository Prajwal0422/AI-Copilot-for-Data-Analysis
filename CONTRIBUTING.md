# Contributing

## Getting Started
1. Fork the repository
2. Clone your fork
3. Create a branch
4. Make changes
5. Commit with clear messages
6. Push and create PR

## Code Style
- Use TypeScript
- Follow Tailwind conventions
- Add comments
- Write clean code

## Commit Messages
- feat: New feature
- fix: Bug fix
- docs: Documentation
- style: Formatting
- refactor: Code restructure
