# Tech Stack

## Frontend
- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React

## Design
- **Style**: Glassmorphism
- **Colors**: Gradient accents
- **Typography**: Inter font
- **Effects**: Backdrop blur, shadows

## Features
- Server-side rendering
- Client-side state management
- Smooth animations (60fps)
- Responsive design
- Modern UI patterns

## Performance
- Optimized animations
- Efficient re-renders
- Fast page loads
- Smooth interactions

## Browser Support
- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)
