# Usage Guide

## Getting Started

### 1. Start the Server
```bash
cd frontend
npm run dev
```

### 2. Open Browser
Navigate to: http://localhost:3000

### 3. Select Dataset
- Click on "sales_data.csv" in the left sidebar
- The card will highlight with a blue gradient

### 4. Send a Message
- Type your question in the input box
- Press Enter or click "Send"
- Watch the workflow animation

### 5. View Response
- AI typing indicator appears
- Workflow steps animate (Query → LLM → Tool → Execute)
- Response appears in chat
